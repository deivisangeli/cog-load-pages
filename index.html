Cog Load private dev pilot

A WhatsApp-native household chief-of-staff for Brazilian middle-class couples. Multi-author research project; build lead Deivis Angeli.

What it is, in one paragraph

Each couple is assigned three agents: two private points-of-contact (one per partner, gender-matched persona) plus a single genderless chief of staff (Pivô / Pivot) that mediates. POCs run "service mode" continuously — capturing tasks, holding lists, sending reminders, anticipating handoffs. Pivot stays out of the way until the data shows asymmetry, then invites reflection and (with both sides' approval) drafts and ferries concrete trade proposals between the two threads. POCs are architecturally incapable of reading each other's chats; only Pivot has cross-partner visibility, and only via structured payloads.

Treatment-mechanism mapping

Each Pivot intervention is tagged with the treatment mechanism(s) it targets, drawn from CogLoad_Treatments.docx:

  • M1 — Information: side-by-side load summary delivered privately to each partner
  • M2 — Norms: descriptive-norm correction relative to similar BR households
  • M4 — Standards: explicit standards-negotiation step before reassignment
  • M5 — Indivisibility: the agent is the externalized household manager
  • M6 — Learning costs: handoff scaffolding (checklists, "ask via me" framing)
  • M9 — Altruism + commitment: structured commitment with explicit duration

Cost envelope

  • 200 couples × 6 months: ~$2,400 total infra (~$12 / couple over 6 months, ~$2 / couple / month)
  • Heavy-user worst case: ~$50 / couple / 6 months
  • Recruitment cost (paid social) likely dominates infra at n ≥ 100
