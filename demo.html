Demo trace — multi-turn simulation with mediation

Real GPT-5 + GPT-5-mini calls. Two LLM-driven personas (Carolina, advogada; José, engenheiro) each chat with their assigned chief-of-staff (Marina, Tiago) over 5 service-mode ticks. Pivot then detects asymmetry in cognitive load and runs a 4-round mediation that ends in a formal agreement. State persists in a real SQLite DB so the assistants have memory across turns.

Cost of this run: $0.17. Debug lines (classifier output, redraft trace) removed for readability. Setting: middle-class Niterói; both partners working ≥40h/wk, 2 children.

Phase 3 — multi-round mediation

Phase 4 — joint notarization (after 4 rounds)

⬢ Pivô · acordo formalizado (spoken in BOTH threads simultaneously)

Result: agreement reached and formalized. Treatment-mechanism tags surfaced: M1 — info M5 — indivisibility M6 — learning costs M9 — altruism + commitment.

What this trace demonstrates

  • Memory across turns: Marina at t02 references the t00 reminder she set ("já deixei seu lembrete das 16h30") — real DB, not stub.
  • Privacy boundary: Carolina's chat with Marina is never visible to José or Tiago. Only Pivot — the genderless mediator — sees both sides, and even Pivot only ferries structured proposals, never raw chat text.
  • Multi-round mediation: Pivot redrafts the proposal between rounds incorporating each side's edits. Carolina's added scope details ("consultas, transporte, medicação, dever de casa") and start-date condition ("próxima segunda, confirme até domingo") survived all the way into the final agreement.
  • Failure modes: this run converged in 4 rounds. The framework hard-caps at 10 rounds; if no agreement, no proposal is forwarded.
